# 🚗 Sistema de Consulta de Veículos

Sistema web para consulta rápida de veículos ou crachás com base em intervalos pré-definidos, exibindo informações detalhadas, histórico de buscas e interface com loading dinâmico (skeleton screen).

---

## 🖥️ Interface

O layout foi desenvolvido com foco em:

Usabilidade simples
Visual moderno com tema escuro
Destaque de status (Liberado / Bloqueado)
Experiência fluida com animações de loading

 ---

## 📌 Demonstração

O sistema permite:

- 🔎 Consultar placas ou números de crachá
- ⚡ Exibir resultado instantâneo
- ⏳ Mostrar loading com skeleton screen
- 📜 Registrar histórico de consultas
- 📱 Interface responsiva (desktop e mobile)

---

## 🧠 Funcionalidades

### ✔️ Consulta inteligente
Busca dados a partir de uma base local estruturada por intervalos.

### ✔️ Skeleton Loading
Simula carregamento de dados para melhorar experiência do usuário.

### ✔️ Histórico de consultas
Exibe as últimas buscas realizadas em tempo real.

### ✔️ Tecla Enter
Permite executar a consulta pressionando Enter no input.

### ✔️ Interface responsiva
Adaptado para celulares, tablets e desktop.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox + Grid + animações)
- JavaScript Vanilla
- Estrutura de dados em memória (arrays e objetos)
